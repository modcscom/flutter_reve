## FlutterReve 介绍

鉴于CloudReve的客户端要收费，并且有手机端的使用需求，本人学了两天flutter就开始上手了。。很多功能还没开发完成，会逐步完善。

鉴于flutter学习时间比较短且作为一个后端程序员，请对代码规范以及使用方式多多包涵

### 实现功能

- [x] 绑定登录
- [x] 站点多用户绑定
- [x] 文件夹浏览
- [x] 基础UI
- [ ] 文件在线查看内容
- [ ] 文件离线查看
- [ ] 文件上传、下载
